## About The Project
<!-- Ïù¥ÎØ∏ÏßÄ ÏÇΩÏûÖ ![Pequod](https://github.com/user-attachments/assets/8bf45ee3-1001-459a-8db2-f32632e20dfc)-->

ü•´ These days, a lot of companies and even countries are trying to get their hands on GPUs. Modern AI needs a ton of GPU power for calculation. But imagine this ‚Äî what if there‚Äôs a large language model that doesn‚Äôt need a GPU at all? Or even run *without* a computer?
I created **AnalogGPT** which is a fully paper-based version of GPT built from scratch.<br/>

üöÄ Via **AnalogGPT** you can fully understand how Generative Pretrained Transformer works by calculating with your bare hands. You are becoming a GPU that calculates these numbers. As the model size is 15K, it's representation capacity is limited. I'll notify the hyper parameters below. Feel free to modify my code and test various things.
